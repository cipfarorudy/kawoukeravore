# Contribuer

Merci de votre intérêt !

## Règles rapides
- Proposez une **issue** avant un gros changement.
- Utilisez des **branches** descriptives : `feature/…`, `fix/…`.
- Respectez le style de code HTML/CSS propre et accessible.

## Processus
1. Fork & clone
2. `git checkout -b feature/ma-fonctionnalite`
3. Faites vos changements (tests manuels)
4. `git commit -m "feat: ma fonctionnalité"`
5. `git push origin feature/ma-fonctionnalite`
6. Ouvrez une **Pull Request**

Merci !
